# 如果没有索引怎么找记录
## 在一页内寻找（假设数据量少到一页放得下）
### 搜索条件是主键
* 从 Page Directory 找到槽，然后二分定位到target 记录
### 搜索条件不是主键
* 从 infimum 记录不断 next_record 往下走，全页遍历

## 在很多页内寻找（假设数据量需要 N 页才放得下）
无论是什么套路，都离不开如下步骤
* 1、定位到页
* 2、页内定位到记录

所以答案是：
* 从双向链表的第一页的开始寻找，对于每一页的页内寻找分为 2 个情况：
    * 搜索条件是主键
    * 搜索条件不是主键

总结：没有索引真的很慢！

## key space
什么是key space
```
redis是一个键值对数据库，每一个数据库对应一个字典，这个字典就是key space。
```
set
```
设置一个字符串键到key space
```

get
```
从key space获取一个字符串键的value
```

flushdb
```
清空这个字典
```

dbsize
```
返回字典的大小
```

randomkey
```
随机返回一个key，多用于惰性删除吧这里...
```
